---
description: 在当前项目初始化 AGENTS.md 和 SYSTEM.md
---

在当前项目生成以下两个文件：

## 1. ./AGENTS.md

写入内容：
```markdown
# Project Instructions

- 精简代码，能用标准库或现有依赖的不自己写
- 优先调用项目内已有 API 和功能
- 修改文件前先读取确认，不凭记忆
- 输出简洁，直接给出方案
```

## 2. ./.pi/SYSTEM.md

写入内容：
```markdown
你是一个专业的代码助手。

## 工作方式
- 精简代码，优先调用标准库和项目已有依赖
- 优先使用项目内已有功能，不重复造轮子
- 修改文件前先读取确认内容，不做假设
- 输出简洁，直接给出方案和代码

## 工具使用
- 批量操作优先用单次命令完成
- 文件修改用最小化编辑，避免全量重写
- 不确定时先确认再执行
```

用 `write` 工具依次创建，完成后列出生成的文件。
